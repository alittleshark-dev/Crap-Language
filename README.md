## 语法

| 语句 | 说明 | 示例 |
| :--- | :---: | :--- |
| `> <内容>;` | 输出语句 | `> "Hello World!";` |
| `[变量] < <提示>;` | 输入语句 | `< user_input;` |
| `<变量名> <值1> <值2>...;` | 列表 | `list 1 1;` |
| `<变量名> <值>;` | 创建变量 | `a 24;` |
| `<条件> ? <真> : <假>;` | 条件判断 (If) | `a = 1 ? > "不是"; : > "是的";; |
| `:<语句>:;` | 死循环 | `a 0; :a+1; a<3? !: ;:;` |
| `!;` | 中断/跳出 (Break) | `!;` |
| `=` | 赋值 (Equal) | `x = 10;` |
| `()` | 括号/表达式分组 | `(1 + 2) * 3;` |
| `;` | 语句结束符 (End) | `> "End";` |
